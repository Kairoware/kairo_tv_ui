# kairo_tv_ui

Uma interface de usuário (UI) moderna e personalizável para Android TV e dispositivos similares, construída com Flutter.

Este projeto serve como um launcher alternativo, focado em um design limpo, com um slideshow de gradientes animados no fundo e acesso rápido aos seus aplicativos favoritos.

## Funcionalidades

- **Fundo Dinâmico:** Slideshow de gradientes que transicionam suavemente.
- **Acesso Rápido:** Uma barra inferior com seus aplicativos favoritos e um botão para a gaveta de aplicativos.
- **Design Responsivo:** A interface se ajusta a diferentes tamanhos de tela.
- **Navegação por Controle Remoto:** O foco é gerenciado para facilitar o uso em TVs.

## Getting Started

This project is a starting point for a Flutter application.

Para executar este projeto, certifique-se de ter o Flutter instalado e configurado.

1. Clone o repositório:
   ```sh
   git clone https://github.com/YourUsername/kairo_tv_ui.git
   ```
2. Entre no diretório do projeto:
   ```sh
   cd kairo_tv_ui
   ```
3. Instale as dependências:
   ```sh
   flutter pub get
   ```
4. Execute o aplicativo:
   ```sh
   flutter run
   ```
